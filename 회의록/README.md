# 회의록 규칙

참고 자료: https://gist.github.com/ihoneymon/652be052a0727ad59601

### <b>규칙</b>

1. 회의록은 .md 확장자로 올리는 것을 지향합니다.
2. 회의에 참여한 사람 모두가 회의록을 작성해야 합니다.
3. 작성한 회의록은 'YYYY-MM-DD-회의록.md' 파일로 PR을 요청합니다.
4. 회의록은 회의한날까지 PR을 요청해야 합니다.
5. 회의 록에는 다음과 같은 내용이 포함되어야 합니다.
6. 작성자, 작성일, 회의 참여자
7. 조사 내용과 조사자
8. 의견과 핵심
9. 업무 배당과 배당 받은 사람, 마감일
10. 회의 결론

### <b> 동작 방식</b>

- 요청된 PR을 발쵀하고 merge하여 하나의 회의록으로 만들어 업로드 됩니다.
